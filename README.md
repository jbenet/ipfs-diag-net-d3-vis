# d3view -- view ipfs network diagnostics in d3

## install

from github
```sh
git clone github.com/jbenet/ipfs-diag-net-d3-vis
cd ipfs-diag-net-d3-vis
sudo make install
```

from ipfs
```sh
coming soon, after we build `ipfs get`
```

## usage

```sh
ipfs diag net --vis=d3 | d3view
```

different viewers:
```sh
ipfs diag net --vis=d3 | d3view chord
ipfs diag net --vis=d3 | d3view force
```
